# Unicol Armenia - Official Website

Official website for Unicol industrial adhesives in Armenia, represented by Roots LLC.

## Overview

This website serves as the official representation of Unicol S.r.l. products in Armenia. Unicol is an Italian company that has been producing industrial adhesives since 1977, offering a wide range of products for various industrial sectors.

## Features

- **Modern Design**: Clean, professional design based on Unicol's brand identity
- **Responsive Layout**: Fully responsive design that works on all devices
- **Multi-language Support**: English and Armenian language support (Armenian coming soon)
- **Product Showcase**: Comprehensive display of Unicol's product lines
- **Application Information**: Details about various industrial applications
- **Contact Form**: Easy way for customers to get in touch
- **Certifications Display**: Showcase of quality and sustainability certifications

## Product Categories

- VINILICHE (Vinyl-based adhesives)
- TERMOFONDENTI EVA E PO (Hot-melt EVA and polyolefins)
- TERMOFONDENTI HMPUR (Hot-melt polyurethane)
- LIQUIDI DISTACCANTI (Release agents)
- DISPERSIONI POLIURETANICHE (Polyurethane dispersions)
- RESINE UREICHE (Urea-formaldehyde resins)
- ADESIVI POLIURETANICI E SILANICI (Polyurethane and silane adhesives)
- AUSILIARI (Auxiliary products)

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required

## Customization

### Updating Contact Information

Edit the contact section in `index.html`:
- Update email addresses
- Update phone numbers
- Update physical address

### Adding Images

Create an `images` folder and add:
- Company logos
- Product images
- Application photos
- Team photos

### Language Support

The website is prepared for Armenian language support. To implement:
1. Create Armenian content translations
2. Update `script.js` language switching functionality
3. Add language-specific content
